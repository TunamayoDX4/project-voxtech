# VoxTech

VoxTechとは3Dサンドボックスゲームの開発を目的としたオープンソースプロジェクトです。
VoxTechは、Minecraftにインスパイアされたゲームエンジンであり、64分木構造を使用して効率的にボクセルデータを管理します。

## 仕様

| | | |
|-|-|-|
| プラットフォーム | Windows, Linux | MacOSはOpenGL/Vulkanをサポートしないため、未対応です。 |
| 言語 | Rust | |
| ライセンス | MIT | ユーザの動向次第でGPLに移行する可能性があります。 |
| グラフィックスAPI | Vulkan, OpenGL | OpenGLは非推奨です。 |
| 物理エンジン | 未定 | |
| サウンドエンジン | rodio | |
| プロセッサアーキテクチャ | AMD64 | AVX512命令セットを使用します |
